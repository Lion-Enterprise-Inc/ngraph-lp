// NFG — Nicomacos Food Graph

料理の意味を、
世界標準で構造化する

NFGは、一つの料理に対して材料・調理法・アレルゲン・宗教制約・味覚プロファイル・カロリーを 8つの構造レイヤーで定義する、世界初の食のデータ規格です。

nfg_schema.json — 越前おろしそば
{
  "dish": "越前おろしそば",
  "name_en": "Echizen Oroshi Soba",
  "allergens": ["小麦", "そば"],
  "dietary": { "vegan": false, "halal": true },
  "ingredients": ["そば粉", "大根", "ねぎ", "鰹節"],
  "taste_profile": {
    "umami": 8, "refreshing": 9, "spicy": 3
  },
  "calorie_range": "350-450 kcal",
  "verified": true  // 店主承認済み (VAD)
}
LAYER 1-3

料理の実体

店舗情報・料理マスタ・185種の材料を網羅。地元食材も構造化データとして保持します。

LAYER 4-6

安全の保証

調理法・食事制約・アレルゲンの3層で「食の安全」を構造化。32品目対応、ハラール・ヴィーガン判定付き。

LAYER 7-8

味の可視化

8次元の味覚プロファイルとカロリーレンジで、AIによるパーソナライズ推薦の精度を向上。

// How It Works

店主の負担は、ほぼゼロ

AIが9割を自動構造化。信頼度の低い箇所だけ、店主が確認・修正する仕組みです。

1

メニューをアップロード

写真・PDF・URLなど、手元にあるメニュー情報をそのまま送るだけ。手入力は不要です。

2

AIが自動構造化

NFG形式に自動変換。材料・アレルゲン・味覚プロファイルまで、AIが一括で生成します。

3

QR設置・運用開始

店主が確認したら、お店専用QRコードを設置。14言語対応のAIコンシェルジュが即日稼働します。

なぜ今、
